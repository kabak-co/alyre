# Alyre, the reading social media
This app help you  keep track of the books you read and rate them. 

## Requirements
React.js - 18.1.0

jest - 5.16.4

tailwind - 3.1.2

## To launch project
Install dependencies

        yarn install


Start the project

        yarn start


Start the test

        yarn test

## To launch Storybook
Run storybook project

        npm run storybook
